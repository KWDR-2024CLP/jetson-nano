# jetson-nano-test


## 주요 개발 내용
1. **웹캠 초기화 및 테스트**
   - OpenCV를 사용하여 Jetson Nano에 연결된 웹캠 피드를  출력합니다.

2. **객체 탐지 통합 (테스트 중)**
   - YOLOv5와 같은 사전 학습된 객체 인식 모델을 활용하여 웹캠 화면에서 객체를 탐지합니다.
